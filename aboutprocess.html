Spotisave

Hi! Welcome to Spotisave.

The Story

Spotify has this amazing feature called "Discover Weekly" and I use it often. I find that I listen to my Discover Weekly playlist more than I listen to any of my favorite artists. That is to say, a lot.

The one caveat is that it gets deleted once a week, every week, to be replaced with something that is probably equally amazing.

When that first happened to me, I started spending a few minutes every week to back up my Discover Weekly into a new playlist. And it was annoying.

I took the opportunity to finally learn about RESTful APIs, and along the development of a script to automate the process, I learned countless other useful skills such as:
  • node.js
  • Gulp
  • Express
  • OAuth
  • HTTP requests
  • Promises
and more. Finally, I decided to make a user-friendly website rather than a more obscure and esoteric command line app. I also took the opportunity to incorporate even more learning experiences into my project, such as:
  • ES2017 (async/await)
  • Babel
  • AJAX requests
  • Browserify
  • UglifyJS, CleanCSS, HTMLMin
  • Challenging myself by creating my own CSS rather than using a library such as Bootstrap, Angular Material (or any other Material Design implementation).
  • SVG and SVG animation
I originally set out to use only vanilla JavaScript as a challenge, but I decided that not using jQuery at the very least would be overly difficult for the original release. Who knows, I might come back and implement my own methods to avoid using the entirety of jQuery. That could take a lot of weight off of the load time.
So Spotisave is more of a learning project or a demo than a business or a start-up. Don't worry, though! It's perfectly functional, so feel free to use it.
